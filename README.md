## Get Started

1. **Clone repository**

   ```shell
   git clone https://github.com/hirebarend/react-admin-mui-template.git
   ```

2. **Install npm packages**

   ```shell
   cd react-admin-mui-template
   npm install
   ```

3. **Start**

   ```shell
   npm start
   ```
